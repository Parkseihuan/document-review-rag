알기 쉬운 행정용어 (요약본)

※ 이 파일은 PDF에서 텍스트로 변환된 예시 파일입니다.
※ 실제 사용 시 국립국어원/행정안전부 제공 자료를 텍스트로 변환하여 대체하세요.

===========================================
1부. 어려운 한자어를 쉬운 말로
===========================================

【ㄱ】
기왕: → 이미, 이전
거주: → 살림, 삶
검토: → 살펴봄, 따져 봄
결재: → 승인
고지: → 알림
공람: → 돌려봄
구비: → 갖춤
귀추: → 앞일, 사태의 진전

【ㄴ】
내역: → 자세한 내용
노후: → 늙어서
농후: → 짙음

【ㄷ】
단속: → 규제, 감시
도래: → 옴, 다가옴
독촉: → 재촉

【ㅁ】
명시: → 밝힘, 분명히 보임
묵인: → 못본 체함
미비: → 갖추지 못함
미흡: → 모자람

【ㅂ】
반영: → 넣음, 담음
방침: → 방향
배제: → 제외, 뺌
변경: → 바꿈
보완: → 보충
불식: → 없앰

【ㅅ】
사전: → 미리
사유: → 까닭, 이유
상기: → 위에 말한
선별: → 가려냄
소급: → 거슬러 올라감
수립: → 세움
시달: → 내려보냄
시정: → 고침
시행: → 실행

【ㅇ】
안건: → 의제, 논의할 문제
여부: → ~인지 아닌지
연유: → 이유
우려: → 걱정
이첩: → 넘김
인지: → 앎
일괄: → 한꺼번에

【ㅈ】
잉여: → 남음, 나머지
재고: → 다시 생각함
전결: → 위임 받아 결정
전반: → 전체, 모두
제고: → 높임
조속: → 빠른
조치: → 처리
준수: → 지킴
지양: → 하지 않으려고 함
지체: → 늦춤
집행: → 실행

【ㅊ】
참고: → 참조
청취: → 들음
촉구: → 재촉
최소: → 가장 적음
추진: → 밀고 나감
충족: → 채움

【ㅍ】
폐지: → 없앰
표기: → 적음, 나타냄

【ㅎ】
하달: → 내려보냄
합의: → 의견을 모음
협조: → 도움
확대: → 늘림
획일: → 한결같음

===========================================
2부. 일본식 한자어를 우리말로
===========================================

일본식 표현 → 순화 표현
- 간담회 → 대화 모임, 담화회
- 검수 → 점검, 확인
- 견적서 → 예산서, 어림셈서
- 납기 → 납품 기한
- 대금 → 값, 돈
- 립서비스 → 말뿐인 약속
- 마감 → 끝냄, 마무리
- 무순 → 순서 없음
- 불황 → 경기 침체
- 사전 준비 → 미리 준비 (사전은 일본식)
- 선착순 → 먼저 온 순서
- 수순 → 차례, 순서
- 유지 → 가짐, 지킴
- 인상 → 늘림, 올림
- 절충 → 조정, 타협
- 주선 → 알선
- 참조 → 참고
- 착수 → 시작
- 취소 → 없앰, 철회

===========================================
3부. 외래어를 우리말로
===========================================

외래어 → 순화어
- 가이드라인 → 지침, 기준
- 노하우 → 기법, 비결
- 디자인 → 설계, 도안
- 리스크 → 위험
- 매뉴얼 → 안내서, 지침서
- 모니터링 → 관찰, 감시
- 백업 → 예비, 보관
- 벤치마킹 → 모범 사례 분석
- 브리핑 → 보고, 설명
- 서포트 → 지원
- 스케줄 → 일정
- 스펙 → 사양, 규격
- 시스템 → 체계, 제도
- 시뮬레이션 → 모의실험
- 아웃소싱 → 외주
- 워크숍 → 실무 교육, 연수회
- 인프라 → 기반 시설
- 컨센서스 → 합의
- 컨설팅 → 자문
- 콘텐츠 → 내용물
- 타깃 → 대상
- 트렌드 → 경향, 추세
- 파트너십 → 협력 관계
- 포커스 → 초점
- 프로세스 → 과정, 절차
- 피드백 → 의견, 반응

===========================================
4부. 공문서 작성 시 자주 틀리는 표현
===========================================

틀린 표현 → 올바른 표현

【띄어쓰기】
- 안됩니다 → 안 됩니다
- 안돼요 → 안 돼요
- 못합니다 → 못 합니다

【맞춤법】
- 되어지다 → 되다
- 이루어지다 → 이루다
- 바래요 → 바라요
- 바램 → 바람
- 갔이 → 같이
- 웬지 → 왠지

【문법】
- ~할려고 → ~하려고
- ~할껀데 → ~할 건데
- ~하실껍니다 → ~하실 겁니다

【공문서 표현】
- ~해 주세요 → ~하여 주시기 바랍니다
- 빠른 시일 내 → 구체적 날짜 명시
- 위와 같이 → 재서술
- ~에 대하여 → ~에 관하여 (간결)
